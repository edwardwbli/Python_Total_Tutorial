def who(n):
    print id(n)
    n=n+1
    print id(n)

看看列表


```python
def who_list(n):
    print id(n)
    n[0]=n[0]+1
    print id(n)
    n.append(10)
    print id(n)
```


```python
b = [1,2,3]
```


```python
id(b)
```




    4360532768




```python
who_list(b)
```

    4360532768
    4360532768
    4360532768



```python
b
```




    [2, 2, 3, 10]



因此我们可以得到结论事实上是传的引用,对于不可变对象,传入的是引用,但这个引用是可以被替换的,也就是实际上它只是把变量指到了值上,但一旦重新赋值,引用就变了,而原来的值并不会变.对于可变对象,变量参数指向传入的对象,对它的改变就是对原来参数的改变
